Informativa sulla privacy - CardScanTrader

CardScanTrader utilizza la fotocamera del dispositivo esclusivamente per scansionare carte Magic: The Gathering selezionate dall’utente.

Dati raccolti

  • Accesso alla fotocamera, usato solo durante la scansione delle carte.
  • Informazioni della carta riconosciute dalle immagini, come nome e set.
  • Token CardTrader fornito dall’utente, salvato localmente sul dispositivo.
  • Dati della coda temporanea salvati localmente sul dispositivo.

Come vengono usati i dati

  • Per scansionare e riconoscere le carte.
  • Per preparare inserzioni per CardTrader.
  • Per pubblicare inserzioni su CardTrader solo quando l’utente sceglie esplicitamente di farlo.

Condivisione dei dati

  • L’app può usare le API di Scryfall per i dati delle carte.
  • L’app può usare le API di CardTrader per la pubblicazione delle inserzioni.
  • L’app non vende dati personali.

Permesso fotocamera

Il permesso della fotocamera è richiesto solo per scansionare le carte all’interno dell’app.

Archiviazione locale

Il token CardTrader e la coda temporanea vengono salvati localmente sul dispositivo dell’utente.

Controllo dell’utente

  • L’utente può rimuovere il token CardTrader dalle impostazioni dell’app.
  • L’utente può eliminare gli elementi in coda direttamente dall’app.

Privacy Policy - CardScanTrader

CardScanTrader uses the device camera only to scan Magic: The Gathering cards selected by the user.

Data collected

  • Camera access, used only during card scanning.
  • Card information recognized from images, such as card name and set.
  • User-provided CardTrader token, stored locally on the device.
  • Temporary queue data stored locally on the device.

How data is used

  • To scan and recognize cards.
  • To prepare listings for CardTrader.
  • To publish listings to CardTrader only when the user explicitly chooses to do so.

Data sharing

  • The app may use Scryfall APIs for card data.
  • The app may use CardTrader APIs for listing publication.
  • The app does not sell personal data.

Camera permission

Camera permission is required only to scan cards inside the app.

Local storage

The CardTrader token and temporary queue are stored locally on the user's device.

User control

  • The user can remove the CardTrader token from the app settings.
  • The user can delete queued items from inside the app.
