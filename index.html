Privacy Policy - CardScanTrader

Ultimo aggiornamento: 7 aprile 2026

CardScanTrader è un'applicazione che consente di scansionare carte Magic: The Gathering e prepararle per la pubblicazione su CardTrader.

Dati gestiti

L’app utilizza esclusivamente i dati necessari al funzionamento:

  • Accesso alla fotocamera, utilizzato solo durante la scansione delle carte.
  • Informazioni sulle carte riconosciute (nome, set, collector number).
  • Token CardTrader inserito dall’utente, salvato localmente sul dispositivo.
  • Coda temporanea delle carte, salvata localmente.

L’app non raccoglie né memorizza dati personali su server dello sviluppatore.

Uso dei dati

  • Riconoscere le carte scansionate.
  • Preparare bozze di pubblicazione.
  • Pubblicare carte su CardTrader solo su richiesta esplicita dell’utente.

Condivisione dei dati

Per funzionare correttamente, l’app comunica con servizi esterni:

  • API Scryfall per ottenere informazioni sulle carte.
  • API CardTrader per pubblicare le carte.

Durante queste operazioni, alcune informazioni (come nome carta, set o token CardTrader) possono essere trasmesse a tali servizi.

L’app non vende né condivide dati personali con terze parti per fini commerciali.

Memorizzazione dei dati

Tutti i dati dell’utente sono salvati localmente sul dispositivo.

Permesso fotocamera

La fotocamera viene utilizzata esclusivamente per la scansione delle carte.

Abbonamenti

L’app può offrire funzionalità premium tramite abbonamento mensile gestito da Google Play.

  • È prevista una prova gratuita di 7 giorni.
  • L’abbonamento si rinnova automaticamente salvo cancellazione.
  • I pagamenti sono gestiti direttamente da Google.
  • Lo sviluppatore non ha accesso ai dati di pagamento dell’utente.

Controllo utente

  • Rimuovere il token CardTrader dalle impostazioni.
  • Eliminare le carte salvate in qualsiasi momento.

Contatti

Email: support.cardscantrader@gmail.com

Privacy Policy - CardScanTrader (English)

Last updated: April 7, 2026

CardScanTrader is an application that allows users to scan Magic: The Gathering cards and prepare them for listing on CardTrader.

Data handled

  • Camera access, used only during card scanning.
  • Recognized card information (name, set, collector number).
  • CardTrader token stored locally on the device.
  • Temporary card queue stored locally.

The app does not collect or store personal data on developer-owned servers.

Data usage

  • Recognize scanned cards.
  • Prepare listing drafts.
  • Publish cards on CardTrader only upon user request.

Data sharing

  • Scryfall API for card data.
  • CardTrader API for publishing.

Some data (such as card name, set or token) may be transmitted to these services.

The app does not sell or share personal data for commercial purposes.

Data storage

All user data is stored locally on the device.

Camera permission

Used only for scanning cards.

Subscriptions

  • A monthly subscription may be offered via Google Play.
  • A 7-day free trial may be available.
  • Subscriptions renew automatically unless canceled.
  • Payments are handled by Google.
  • The developer does not access payment data.

User control

  • Users can remove their CardTrader token at any time.
  • Users can delete stored cards at any time.

Contact

Email: support.cardscantrader@gmail.com
