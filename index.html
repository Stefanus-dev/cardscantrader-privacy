Privacy Policy - CardScanTrader

Ultimo aggiornamento: 2 aprile 2026

CardScanTrader utilizza la fotocamera del dispositivo esclusivamente per consentire all’utente di scansionare carte Magic: The Gathering.

Dati raccolti

  • Accesso alla fotocamera, usato solo durante la scansione.
  • Informazioni sulle carte riconosciute (nome, set, ecc.).
  • Token CardTrader inserito dall’utente, salvato localmente.
  • Coda temporanea delle carte, salvata localmente.

Come vengono usati i dati

  • Per riconoscere le carte scansionate.
  • Per preparare bozze di pubblicazione.
  • Per pubblicare su CardTrader solo su richiesta dell’utente.

Condivisione dei dati

  • Uso di Scryfall per dati sulle carte.
  • Uso API CardTrader per pubblicazione.
  • Nessuna vendita di dati personali.

Memorizzazione locale

I dati restano salvati sul dispositivo dell’utente.

Permesso fotocamera

Usato solo per la scansione delle carte.

Controllo utente

  • Rimozione token possibile dalle impostazioni.
  • Eliminazione carte dalla coda possibile.

Privacy Policy (English)

Last updated: April 2, 2026

CardScanTrader uses the device camera only to scan Magic: The Gathering cards.

Data collected

  • Camera access used only during scanning.
  • Card data such as name and set.
  • CardTrader token stored locally.
  • Temporary queue stored locally.

How data is used

  • To recognize scanned cards.
  • To prepare listings.
  • To publish on CardTrader when requested.

Data sharing

  • Scryfall API for card data.
  • CardTrader API for publishing.
  • No personal data is sold.

Local storage

Data is stored locally on the device.

Camera permission

Used only for scanning cards.

User control

  • User can remove token anytime.
  • User can delete stored cards.
